【核心工作流】
═══════════════════════════════════════════════════════════════════════════

  1️⃣  准备视频（命名规范见下方）
      ↓
  2️⃣  添加到 data/input/
      ← bash scripts/add_video.sh /path/to/video.mp4  （推荐）
      ← 或直接拖入文件夹
      ↓
  3️⃣  执行流水线
      ← bash scripts/run_pipeline.sh
      ↓
      ├─ Step 1  提取媒体      src/extract_media.py
      │           输入：data/input/*.mp4
      │           输出：data/output/{视频名}/audio.mp3
      │                  data/output/{视频名}/frame_*.jpg
      │
      ├─ Step 2  转录音频      src/transcribe_audio.py
      │           使用 Groq Whisper API（增量，已完成自动跳过）
      │           大文件 >18MB 自动分片（5分钟/片）
      │           输出：data/output/{视频名}/transcript/audio.srt
      │                  data/output/{视频名}/transcript/transcript.md
      │           聚合：data/srt_exports/{课程名}/{段落名}.srt
      │
      ├─ Step 3  生成笔记      src/generate_note.py --all
      │           使用 Groq LLM（llama-3.3-70b）
      │           同课程多视频段自动合并 → 一份课程笔记
      │           笔记结构：Mind Map + 数据表格 + 分段详情
      │           输出：notes/{课程名}.md
      │                  notes/INDEX.md（自动更新）
      │
      └─ Step 4  清理临时文件  rm -rf data/output/*/transcript/_chunks_work/
      ↓
  4️⃣  在 Obsidian 中查看
      └─ notes/{课程名}.md
      └─ notes/INDEX.md

【命名规范】
═══════════════════════════════════════════════════════════════════════════

  格式：{课程编号}_{课程名}_{段编号}_{段标题}.mp4

  ✅ 00_03_第三课：示例课程丙_01_第一节内容.mp4
  ✅ 00_03_第三课：示例课程丙_02_第二节内容.mp4
     → 生成笔记：notes/00_03_第三课：示例课程丙.md

  ❌ 视频.mp4           （无法识别课程结构）
  ❌ lesson03_v3.mp4    （无课程编号格式）


【快速命令速查】
═══════════════════════════════════════════════════════════════════════════

┌─ 📥 添加视频 ──────────────────────────────────────────────────────────┐
│                                                                          │
│  bash scripts/add_video.sh /path/to/video.mp4        # 添加 + 自动处理  │
│  bash scripts/add_video.sh --no-run /path/to/v.mp4   # 仅添加，延后处理 │
│                                                                          │
└──────────────────────────────────────────────────────────────────────────┘

┌─ 🚀 执行流水线 ────────────────────────────────────────────────────────┐
│                                                                          │
│  bash scripts/run_pipeline.sh                    # 标准执行（增量）      │
│  bash scripts/run_pipeline.sh --force-note       # 强制重新生成笔记      │
│  bash scripts/run_pipeline.sh --force-transcribe # 强制重新转录          │
│  bash scripts/run_pipeline.sh --note-only        # 仅生成笔记            │
│  bash scripts/run_pipeline.sh --skip-extract     # 跳过提取              │
│                                                                          │
└──────────────────────────────────────────────────────────────────────────┘

┌─ 🔧 单步执行 ──────────────────────────────────────────────────────────┐
│                                                                          │
│  .venv/bin/python src/extract_media.py                   # 仅提取        │
│  .venv/bin/python src/transcribe_audio.py                # 仅转录（增量）│
│  TRANSCRIBE_FORCE=true .venv/bin/python src/transcribe_audio.py  # 强制 │
│  .venv/bin/python src/generate_note.py --all             # 生成所有笔记  │
│  .venv/bin/python src/generate_note.py --prefix "00_03_第三课：示例课程丙"│
│  .venv/bin/python src/generate_note.py --update-index    # 仅重建索引    │
│                                                                          │
└──────────────────────────────────────────────────────────────────────────┘

┌─ 📖 在 Cursor 中引用 SRT 转录 ────────────────────────────────────────┐
│                                                                          │
│  @data/srt_exports/00_03_第三课：示例课程丙/    # 整课所有 SRT          │
│  @data/srt_exports/00_01_第一课：示例课程甲/    # 整课所有 SRT          │
│                                                                          │
└──────────────────────────────────────────────────────────────────────────┘


【关键特性】
═══════════════════════════════════════════════════════════════════════════

  ✅ 增量处理
     已提取 / 已转录 / 已生成笔记 → 自动跳过
     --force 或 FORCE=true 可覆盖

  ✅ 自动分片
     音频 >18MB 自动分片（5分钟/片），处理 API 大小限制

  ✅ 课程自动合并
     同前缀视频段（01/02/03...）自动合并为一份课程笔记

  ✅ SRT 聚合导出
     转录完成后自动聚合到 data/srt_exports/，按课程分类

  ✅ 结构化笔记
     Mind Map（markmap）+ 数据表格 + 分段详情，Obsidian 兼容

  ✅ INDEX.md 自动维护
     每次生成笔记后更新，包含课程统计
